Order Management & Search API

Scalable backend API using FastAPI & MongoDB

Overview

This project is a backend service built using FastAPI that enables order management and advanced search functionality. It is designed with clean architecture principles and is suitable for scalable production systems.

Key Features

  • Create, update, and list orders
  • GST calculation with accurate totals
  • Search orders by item name and price
  • Modular router and service-based architecture
  • RESTful API design

Technology Stack

  • Python
  • FastAPI
  • MongoDB
  • REST APIs

Project Structure

  • routers/ – API routes
  • services/ – Business logic
  • models/ – Database models
  • schemas/ – Request/response schemas
  • utils/ – Utilities and helpers
